About This Project

Universal Media Downloader is a lightweight yet powerful web tool built with Python (Flask) and yt-dlp that allows you to download videos and audio from multiple platforms like YouTube, Instagram, Facebook, Twitter, and more.

✨ Features:

  • Multiple format & resolution options
  • Private video downloads with cookies.txt
  • Download history tracking
  • Dark mode support
  • Batch download support
